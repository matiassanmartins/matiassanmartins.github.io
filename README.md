# Sitios Webs Estaticos
En este apartado encontrarás sitios web donde busco mejorar constantemente mis habilidades en desarrollo web y programación.

Aunque el diseño estético y visual no es mi punto más fuerte, me enfoco en construir aplicaciones claras, funcionales y con lógica sólida. Me interesa más la parte técnica, la resolución de problemas y la experiencia de funcionamiento que el aspecto artístico del sitio.

🚀 Tecnologías que uso
* HTML5
* CSS3
* JavaScript
* Git & GitHub

🎯 Objetivo

Seguir aprendiendo y desarrollando proyectos que me permitan mejorar tanto en frontend como en programación general, explorando nuevas herramientas y buenas prácticas.

“Prefiero un sitio simple que funcione bien, antes que uno bonito que no haga nada.”
